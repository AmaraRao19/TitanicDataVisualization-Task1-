# Titanic EDA Project

## Task Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand patterns, trends, and relationships between different features. The goal is to extract meaningful insights using data visualization techniques and basic data analysis.


## Dataset Used

* **Dataset:** Titanic Dataset
* **Source:** Commonly available dataset used in machine learning and data analysis (Kaggle Titanic Dataset)
* **Description:**
  The dataset contains information about passengers aboard the Titanic, including:

  * Age
  * Gender
  * Passenger Class (Pclass)
  * Fare
  * Embarked Location
  * Survival Status


## Models Applied

* Linear Regression
* This project focuses on:

  * Data Cleaning
  * Data Exploration
  * Data Visualization


##  Key Results and Findings

* **Gender Impact:** Females had a significantly higher survival rate compared to males.
* **Passenger Class:** Passengers in higher classes (1st class) had better chances of survival.
* **Age Factor:** Younger passengers had slightly higher survival rates.
* **Fare Relation:** Higher ticket fares were generally associated with higher survival probability.
* **Embarkation Point:** Survival rates varied depending on the port of embarkation.


## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn


## Conclusion

This analysis highlights how different factors such as gender, class, and fare influenced survival on the Titanic. The project demonstrates the importance of **EDA in understanding datasets before applying machine learning models**.
